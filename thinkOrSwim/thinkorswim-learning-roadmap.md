# ThinkOrSwim Platform - Structured Learning Roadmap

## Overview
This guide provides a step-by-step approach to mastering the ThinkOrSwim platform, from basic navigation to advanced trading strategies.

---

## Phase 1: Foundation (Week 1-2)

### Module 1.1: Platform Setup & Navigation
**Time: 2-3 hours**

- **Getting Started**
  - Download and install ThinkOrSwim desktop application
  - Paper trading account setup (practice without real money)
  - Understanding the main interface layout
  - Customizing workspace and saving layouts

- **Key Interface Components**
  - Menu bar and top toolbar
  - Watchlists panel
  - Charts panel
  - Trade tab
  - Monitor tab
  - MarketWatch panel

- **Practice Exercise**: Create and save 3 different workspace layouts for different purposes (e.g., day trading, options analysis, research)

### Module 1.2: Watchlists & Market Data
**Time: 2 hours**

- Creating and organizing watchlists
- Adding/removing symbols
- Understanding quote columns and customizing them
- Real-time vs. delayed data
- Using the symbol search function
- Creating custom quote columns with thinkScript basics

- **Practice Exercise**: Build a watchlist of 10-15 stocks you want to track, customize columns to show key metrics

### Module 1.3: Basic Charting
**Time: 3-4 hours**

- Chart types (candlestick, bar, line, Renko, etc.)
- Time frames (intraday, daily, weekly, monthly)
- Adding multiple time frames to one view
- Basic drawing tools (trendlines, horizontal lines, Fibonacci)
- Saving chart templates
- Grid and flex grid layouts

- **Practice Exercise**: Create charts for different strategies (scalping, swing trading, long-term investing)

---

## Phase 2: Intermediate Skills (Week 3-4)

### Module 2.1: Technical Analysis Tools
**Time: 4-5 hours**

- **Moving Averages**
  - Simple (SMA) and Exponential (EMA)
  - Common periods (20, 50, 200)
  - Multiple moving average strategies

- **Momentum Indicators**
  - RSI (Relative Strength Index)
  - MACD (Moving Average Convergence Divergence)
  - Stochastic oscillator
  - Understanding overbought/oversold conditions

- **Volume Indicators**
  - Volume bars and interpretation
  - Volume-weighted average price (VWAP)
  - On-balance volume (OBV)

- **Volatility Indicators**
  - Bollinger Bands
  - ATR (Average True Range)
  - Standard deviation channels

- **Practice Exercise**: Set up 3 charts with different indicator combinations and backtest their effectiveness on historical data

### Module 2.2: Advanced Charting Features
**Time: 3-4 hours**

- Studies and strategies overlay
- Customizing study parameters
- Using multiple studies together
- Chart bubbles and labels
- Pattern recognition tools
- Chart sharing and collaboration
- Comparison charts (comparing multiple stocks)

- **Practice Exercise**: Create a complete technical analysis setup for your trading style

### Module 2.3: Scanner Tool
**Time: 3-4 hours**

- Understanding the Stock Hacker (scanner)
- Pre-built scan queries
- Creating custom scans
- Common scan criteria:
  - Volume spikes
  - Price breakouts
  - Moving average crossovers
  - Gap up/down scans
  - Unusual options activity
- Saving and organizing scans
- Running scans at different times

- **Practice Exercise**: Build 5 custom scans for different trading opportunities (momentum, breakouts, value, etc.)

---

## Phase 3: Trading Execution (Week 5-6)

### Module 3.1: Order Types & Execution
**Time: 4-5 hours**

- **Basic Order Types**
  - Market orders
  - Limit orders
  - Stop orders
  - Stop-limit orders

- **Advanced Order Types**
  - Trailing stops
  - Bracket orders (OCO - One Cancels Other)
  - Conditional orders
  - Time-in-force options (DAY, GTC, GTD)

- **Trade Tab Functions**
  - Quick order entry
  - Active trader ladder
  - Order confirmation and modification
  - Order status monitoring

- **Risk Management**
  - Position sizing calculator
  - Setting stop losses and profit targets
  - Understanding bid-ask spread and slippage

- **Practice Exercise**: Execute 20+ paper trades using different order types to understand execution

### Module 3.2: The Analyze Tab
**Time: 3-4 hours**

- Understanding the risk profile graph
- Analyzing profit/loss scenarios
- Break-even points
- Greeks visualization (Delta, Gamma, Theta, Vega)
- Probability analysis
- Add simulated trades
- What-if scenarios (changing price, volatility, time)

- **Practice Exercise**: Analyze 10 different trade scenarios before execution

### Module 3.3: Monitor Tab & Account Management
**Time: 2-3 hours**

- Activity and Position monitoring
- Understanding P&L calculations
- Unrealized vs. realized gains/losses
- Account statement interpretation
- Order history and analysis
- Working orders management

- **Practice Exercise**: Review and analyze all your paper trades to identify patterns in wins/losses

---

## Phase 4: Options Trading (Week 7-9)

### Module 4.1: Options Basics in TOS
**Time: 5-6 hours**

- Options chain navigation
- Strike selection
- Expiration dates
- Bid/ask spreads in options
- The Greeks explained:
  - Delta (directional risk)
  - Gamma (delta acceleration)
  - Theta (time decay)
  - Vega (volatility sensitivity)
  - Rho (interest rate sensitivity)
- Implied volatility vs. historical volatility
- Probability of profit (POP)

- **Practice Exercise**: Study options chains for 10 different stocks across various market conditions

### Module 4.2: Basic Options Strategies
**Time: 6-8 hours**

- **Single-Leg Strategies**
  - Long calls
  - Long puts
  - Covered calls
  - Cash-secured puts

- **Spread Strategies**
  - Vertical spreads (bull/bear call/put spreads)
  - Calendar spreads
  - Diagonal spreads
  - Iron condors
  - Butterflies
  - Straddles and strangles

- Using the Trade tab for multi-leg orders
- Strategy rolls and adjustments
- Closing positions early

- **Practice Exercise**: Paper trade 5 different options strategies and analyze their performance

### Module 4.3: Advanced Options Tools
**Time: 4-5 hours**

- **Sizzle Index** (unusual options activity)
- **Probability Analysis** tools
- **Risk Profile** customization
- **Portfolio margin** calculations
- **Strategy Roller** for managing positions
- Custom options scans
- Volatility smile and skew analysis

- **Practice Exercise**: Use advanced tools to identify and analyze 10 high-probability options trades
