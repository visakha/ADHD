## Get Running in 5 Minutes

### Step 1: Install (Choose Your Platform)

**Linux/macOS:**
```bash
chmod +x install.sh
./install.sh
```

**Windows:**
```cmd
install.bat
```

### Step 2: Get API Key
1. Go to https://console.anthropic.com
2. Sign up / Log in
3. Click "API Keys" â†’ "Create Key"
4. Copy the key (save it somewhere safe!)

### Step 3: Run the App

**Linux/macOS:**
```bash
./run.sh
```

**Windows:**
```cmd
run.bat
```

**OR manually:**
```bash
# Activate venv first
source venv/bin/activate  # Linux/macOS
```

1. App opens
2. Go to `File > Settings`
3. Paste your API key
4. Click "Save"

2. Give it an exciting title
3. Describe why it matters to you
4. Rate your enthusiasm (be honest!)

## Common First-Time Issues

**"Module not found: anthropic"**
```bash
pip install -r requirements.txt
```

**"API Error"**
- Double-check your API key in Settings
- Make sure you have internet connection
- Verify key at console.anthropic.com

**"Can't connect to Claude"**
- Check internet connection
- Verify API key is correct
- Check Anthropic service status

**"TKinter error" (Linux only)**
```bash
sudo apt-get install python3-tk
```
